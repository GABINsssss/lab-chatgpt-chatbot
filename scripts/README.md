# messages_to_langchain.py 사용법

필요 패키지 설치:

```bash
pip install -r requirements.txt
```

환경 변수 설정 (예: `.env` 파일):

- `OPENAI_API_KEY` : OpenAI API 키
- `MODEL` (선택) : 사용할 모델 (기본: `gpt-4o-mini`)

실행:

```bash
python scripts/messages_to_langchain.py
```

종료: 입력창에 `/quit` 입력

에러가 나는 경우: 패키지 설치, 환경변수, 인터넷 연결을 확인하세요.
